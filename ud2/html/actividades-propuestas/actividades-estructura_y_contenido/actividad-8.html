Clasificación de elementos: Estilo vs Semántica

Elementos de estilo (no semánticos)

Este es un texto con cursiva usando <i>, que solo cambia la apariencia sin agregar significado. También, aquí está negrita usando <b>, que solo resalta visualmente.

Etiquetas semánticas

Este es un texto con énfasis usando <em>, que agrega significado al contenido, indicando énfasis. Además, este texto tiene importancia usando <strong>, que semánticamente destaca algo crucial.
