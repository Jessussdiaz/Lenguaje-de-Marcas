¿Qué sucede al probar estos enlaces?

Desde un Pc

mailo:Al hacer clic, se abre la aplicación de correo predeterminada (como Outlook) si tienes una configurada

tel:En un navegador de escritorio, al hacer clic generalmente no se realiza ninguna acción o se muestra un mensaje de error si el dispositivo no tiene la capacidad de realizar llamadas.


Desde un movil

mailo:Al hacer clic, se abre la aplicación de correo predeterminada (como Gmail) crea un nuevo correo con la dirección ya incluida

tel:Al hacer clic, se abre la aplicación de llamadas del teléfono y el número está listo para ser marcado, permitiéndote llamar directamente.
